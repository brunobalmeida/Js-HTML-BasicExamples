Example 3 - Javascript loop 

Create a web page that will display a countdown to a rocket blasting off.

1. HTML Content 
Create a web page that contains a button and an image of a rocket ship.

2. When the user presses the button, a countdown should display on-screen.

3. The countdown should begin at 10 and decrement by integer steps of 1. 

4. When the countdown reaches 0, the display should output "BLASTOFF!" instead of
the number 0. Do not continue the countdown after this point.
