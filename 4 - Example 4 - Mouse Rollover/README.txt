Example 4 - Mouse Rollover

Create a web page that demonstrates the onmouseover and onmouseout events. 

1. HTML Static Content 
Create a web page that contains an image.

2. JavaScript Event - Mouseover 
When the user moves the mouse pointer over the image, change the image that is
displayed.

3. JavaScript Event - Mouseout 
When the user moves the mouse pointer o of the new image, change the image that is
displayed back to the original.
